## このレポジトリの存在意義

このレポジトリの主目的は、zennの記事（）で語られているCursorのProjectRulesの運用イメージを伝えることです。

実際のファイルの中身に関しては、先人たちが公開している優れた内容をいいとこ取りしています。
なので、どなたでも使える内容になっていますがプロジェクトによって大きくチューニングする必要があることを認識した上で慎重にご使用ください。
